This is a lightweight desktop-based Countdown Timer application developed using Python and Tkinter. The app allows users to set custom minutes and seconds, start the timer, pause it anytime, and reset it easily. It features a clean graphical interface and provides a notification popup when the countdown finishes.
Features:
Set custom minutes and seconds
Start, Pause, and Reset controls
Real-time countdown display (MM:SS)
Popup alert when the timer ends
Simple and clean GUI
Beginner-friendly Python project

Tech Stack:
Python
Tkinter (GUI Library)
